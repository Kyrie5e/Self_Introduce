# Self_Introduce

一个基于原生 `HTML + CSS + JavaScript` 实现的个人介绍页面项目，适合用作简历外链、作品展示页和个人主页。

项目地址：
[https://github.com/Kyrie5e/Self_Introduce](https://github.com/Kyrie5e/Self_Introduce)

## 项目简介

这个页面围绕“个人介绍 + 滚动履历展示”来设计，包含首屏聚焦展示、固定导航、分区卡片、翻转弹窗、联系方式快捷交互，以及中英文切换功能。

首页采用探照灯式双层视觉效果：
- 顶层显示中文文案
- 底层显示英文文案
- 鼠标移动时，通过圆形遮罩露出底层内容
- 首屏文字会跟随鼠标产生较强的视角偏转

后续内容采用滚动分区展示：
- 关于
- 技能
- 项目
- 实习
- 校园
- 证书
- 联系方式

其中大部分信息以圆角卡片形式呈现，点击后通过翻转弹窗查看详情；联系方式卡片则改为直接复制或跳转，不再弹窗。

## 页面特点

- 全屏首屏展示，支持探照灯 reveal 效果
- 固定顶部导航，点击可跳转到对应模块
- 卡片式履历展示，支持鼠标视角偏转
- 弹窗采用左右翻转进入和退出动画
- 联系方式卡片支持点击复制邮箱、复制微信、跳转 GitHub
- 右上角支持 `CN / EN` 中英文切换
- 语言切换状态会保存在浏览器本地
- 无需构建工具，打开即可运行

## 本地预览

这是一个纯静态项目，不依赖打包工具。

### 方式一：直接打开

直接双击 `index.html` 即可在浏览器中查看。

### 方式二：使用本地静态服务器

如果你希望本地开发体验更稳定，可以使用任意静态服务器，例如：

```bash
npx serve .
```

或：

```bash
python -m http.server 8080
```

然后在浏览器中访问对应地址。

## 当前功能说明

### 1. 首屏交互

- 中文与英文双层叠加
- 鼠标区域显示底层内容
- 光标为圆形探照灯
- 仅文字内容产生视角偏转

### 2. 履历内容区

- 每个模块用卡片划分
- 大部分卡片点击后显示详细弹窗
- 弹窗为左右翻转动画

### 3. 联系方式

- 邮箱：点击自动复制
- 微信：点击自动复制
- GitHub：点击自动跳转

### 4. 中英文切换

- 右上角胶囊按钮切换 `CN / EN`
- 导航、首屏、卡片、弹窗、联系方式提示会同步切换

## 如果你要继续改这个项目

你通常只需要关注这 3 个文件：

- `index.html`
  用来调整页面结构、导航、静态挂点

- `style.css`
  用来调整主题、卡片尺寸、动画、布局和按钮样式

- `script.js`
  用来修改履历数据、弹窗逻辑、联系方式交互和中英文文案
